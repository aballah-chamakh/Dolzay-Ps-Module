# Dolzay – PrestaShop Carrier Sync Plugin

Dolzay is a PrestaShop module that automatically synchronizes your store orders with your carrier's platform. This plugin allows merchants to streamline their order management by sending order information to the carrier and keeping order statuses in sync between the PrestaShop store and the carrier platform.

## Features

- **Automatic Order Submission**: Sends new orders from your PrestaShop store directly to the carrier platform.  
- **Order Status Synchronization**: Updates the status of orders in your PrestaShop store based on the status of the corresponding order on the carrier platform.  
- **Seamless Integration**: Works in the background without manual intervention, reducing errors and saving time.  

## Installation

1. Download or clone this repository.  
2. Copy the module folder into the `/modules` directory of your PrestaShop installation.  
3. Go to the PrestaShop admin panel → `Modules` → `Module Manager`.  
4. Locate Dolzay and click `Install`.  
5. Configure your carrier credentials in the module settings.  

## Usage

- Once installed and configured, Dolzay automatically sends new orders to the carrier.  
- Order statuses are updated in your PrestaShop store whenever there is a status change on the carrier platform.  
- No manual syncing required.

## Requirements

- PrestaShop 1.7.x  
- PHP 7.4 or higher  
- Access to the carrier API
