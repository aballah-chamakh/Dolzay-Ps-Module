# Dolzay â€“ PrestaShop Carrier Sync Plugin

Dolzay is a PrestaShop module that automatically synchronizes your store orders with your carrier's platform. This plugin allows merchants to streamline their order management by sending order information to the carrier and keeping order statuses in sync between the PrestaShop store and the carrier platform.
